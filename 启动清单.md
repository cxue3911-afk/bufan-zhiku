# 不凡科技运营助手启动清单

## 第一步：补充基础资料

建议优先把以下资料放入 `参考资料/`：
- 公司介绍
- 产品介绍
- 目标用户画像
- 当前社区/产品所处阶段
- 领导对市场营销部的核心要求
- 近期重点项目
- 竞品或参考社区案例

## 第二步：补充工作模板

如果你已有内部模板，请放入 `模板库/`：
- 日报模板
- 周报模板
- 活动方案模板
- 复盘模板
- 领导汇报模板
- 专家邀请函模板

## 第三步：开始日常协作

每天结束工作时，对 Agent 说：

```text
帮我写今天日报：……
```

每周结束时，对 Agent 说：

```text
根据本周日报，帮我生成周报。
```

每次策划活动时，对 Agent 说：

```text
帮我策划一个活动，目标用户是……，活动目标是……，希望达到……
```

## 第四步：让 Agent 自我优化

当你觉得某个输出很好，可以说：

```text
这个结构以后作为默认模板。
```

当你觉得某个输出不符合你的风格，可以说：

```text
以后不要这样写，改成……
```

Agent 会将稳定、可复用的要求沉淀进 `AGENTS.md`。

## 第五步：推荐的第一批输入

你可以先给 Agent 这几类信息：

1. 不凡科技是做什么的
2. 你们当前核心用户是谁
3. 你最近一个月的主要目标
4. 你领导最关注哪些结果
5. 你常做的活动类型
6. 你希望日报更偏简洁还是更偏详细
